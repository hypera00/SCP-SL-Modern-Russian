## Как скачать и использовать:

1. Скачайте ZIP-архив с переводом

2. Распакуйте содержимое архива в папку с игрой по следующему пути:
<i>Steam\steamapps\common\SCP Secret Laboratory\Translations</i>

<b>Как найти папку:</b> зайдите в библиотеку Steam, нажмите правой кнопкой мыши на SCP:SL, перейдите в раздел «Управление», выберите «Просмотреть локальные файлы», а затем откройте папку с названием Translations

3. Запустите игру и в настройках измените язык на <b>Russian (mod)</b>
