<b>Как скачать и использовать:</b>

1. Скачайте ZIP-архив с переводом

2. Распакуйте содержимое архива в папку с игрой по следующему пути:
<i>Steam\steamapps\common\SCP Secret Laboratory\Translations</i>

3. Как найти папку: зайдите в библиотеку Steam, нажмите правой кнопкой мыши на SCP:SL, перейдите в раздел «Управление», выберите «Просмотреть локальные файлы», а затем откройте папку с названием Translations

4. Запустите игру и в настройках измените язык на <b>Russian (mod)</b>
